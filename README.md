# RocoArena · 洛克王国模拟对战系统

RocoArena 是一个 **洛克王国模拟对战项目**，目标是构建一个
**低延迟、可复现战斗机制的对战与研究平台**。

项目当前仍处于早期阶段，主要精力集中在 **战斗机制的验证与实现**。
如果后续进展顺利，将考虑提供 Web 形式供玩家体验与测试。

---

## 项目状态说明

> ⚠ **早期开发 / 实验阶段**

- 机制与数值仍在不断研究与修正中
- 不保证与当前正式服 100% 一致
- 不保证长期维护或稳定接口

若你对洛克王国的战斗系统、数值模拟或对战框架实现感兴趣，
欢迎交流或参与讨论。

---

## 当前待解决问题（Roadmap）

- [ ] **暴击率机制确认**
  - 基础暴击率
  - 装备加成方式
  - 是否存在隐藏修正

- [ ] **命中率 / 回避率公式**
  - 命中、闪避、强化之间的关系
  - 边界与上限情况

- [ ] **回避强化面板实现**

- [ ] 战斗流程与状态结算稳定化
